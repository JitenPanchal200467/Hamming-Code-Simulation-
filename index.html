Origin

Richard Hamming, 1950

Invented at Bell Labs to automatically detect and correct single-bit errors in digital data transmitted over noisy channels — a revolution in reliable computing.

02
Parity Bits

Redundancy Formula

For m data bits, you need r parity bits where 2r ≥ m + r + 1. For 4 data bits → 3 parity bits → 7-bit codeword.

2ʳ ≥ m + r + 1
03
Detection

Syndrome Calculation

XOR-check all parity groups. A non-zero syndrome value pinpoints the EXACT bit position of the error for single-bit correction.

04
Correction

One-Flip Fix

Flip the bit at the syndrome position. SECDED (SEC-DED) variants can detect 2-bit errors and correct 1-bit, used in ECC RAM today.

Full Algorithm — Step by Step
STEP 1

Identify Parity Positions

Powers of 2 (1, 2, 4, 8…) are parity bit positions. All other positions carry actual data bits. Place data bits in non-power-of-2 slots.

STEP 2

Calculate Parity Bits

Each parity bit Pₖ = XOR of all bits whose position has bit k set in binary. Makes each group's XOR = 0 (even parity).

STEP 3

Transmit Codeword

Send the full 7-bit codeword. During transmission over a noisy channel, any single bit may get flipped from 0→1 or 1→0.

STEP 4

Compute Syndrome

Receiver recomputes all parity checks. A non-zero syndrome (S₁S₂S₄ in binary) equals the error position. Flip that bit to correct!
